Markdown 学习
----------
mark是一种轻量级标记语言，他允许人们使用易读的纯文本格式编写代码，然后转化为有效的html文档

##markdown的优点
1. 专注于文字而不是排班样式
2. 轻松的导出html、pdf和本身的.md文件
3. 纯文本内容，支持所有的文本编辑器和文字处理器
4. 可读、直观。适合所有人的写作语言。
